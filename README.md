# DashAnimatedCSS

## notes

To make your element perform a "hinge" animation, you just need to replace animate__bounce with animate__hinge in the className parameter. This will apply the hinge animation from the Animate.css library.

## General Animation

### Bounce
> animate__bounce

### Flash
> animate__flash

### Pulse
> animate__pulse

### Rubberband
> animate__rubberBand
